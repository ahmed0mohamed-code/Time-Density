# time-density- README

An extension that tracks your typing rate in wor.ds per minutes (WPM) and give points depending on your coding rate and changes the color with passing the average rate.

## Features

Calculate the time, Words per minute, characters per minute, number of words, and give you points on them as shown in the following Figure.
